# Gratis

A hosting control panel that doesn't charge you like it's your fault they have investors.

Gratis is a modern, open-source replacement for cPanel — built by someone who got tired of paying for stagnation. It's API-first, supports more than just PHP, and is free to run on your own infrastructure.

## Features

> **Status: early development.** The list below is the target, not the current state.

- **Web hosting management** — domains, subdomains, DNS, SSL (Let's Encrypt), file manager
- **Multi-runtime support** — PHP (multiple versions), Node.js, Python, Ruby — not just PHP
- **Email** — accounts, forwarders, autoresponders, DKIM/SPF/DMARC automation
- **Databases** — MySQL/MariaDB and PostgreSQL, per-account users
- **Cron jobs, FTP accounts, backups**
- **Git deployment** built in
- **Modern UI** — actually pleasant to use
- **Full REST API** — integrate with any billing system or automate anything
- **Multi-server** — manage a fleet from one place

## Philosophy

cPanel raised prices 300% in 2019 and has shipped little of substance since. The alternatives are either expensive, ugly, or both. Gratis exists because the problem is solved — it just costs too much.
